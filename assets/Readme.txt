Zro Dfects – Tetris & Dr Mario Style Puzzle Tileset (Complete Pack)

A huge thank you for downloading my pixel tile asset pack — I really appreciate your interest and support, it really means a lot and highly appreciated.

🎨 What’s Included:
- Screen/canvas size is 320x180, perfect for when zooming on HD ready screens.
- 16 solid pattern Tetris style block shapes in 8 different colors, all in 8x8 tiles, designed for seamless connections
- 16 hollow pattern Tetris style block shapes in 8 different colors, all in 8x8 tiles, designed for seamless connections
- Dr Mario style block shapes in 8 different colors, all in 8x8 tiles, designed for seamless connections
- 16 solid pattern Tetris style block shapes in 1bit black on white, all in 8x8 tiles, designed for seamless connections
- 16 hollow pattern Tetris style block shapes in 1bit white on black, all in 8x8 tiles, designed for seamless connections
- 16 hollow pattern Tetris style block shapes in 1bit green on white, all in 8x8 tiles, designed for seamless connections
- Dr Mario style block shapes with 5 different 1bit white pattern, all in 8x8 tiles, designed for seamless connections
- Dr Mario style block shapes with 5 different 1bit black pattern, all in 8x8 tiles, designed for seamless connections
- Dr Mario style block shapes with 5 different 1bit green pattern, all in 8x8 tiles, designed for seamless connections
- All Tetris style blocks have a 4 directional rotated tile set
- All Dr Mario style blocks have a 2 directional rotated tile set 
- Both Tetris and Dr Mario style games have a dotted drop block place guidance marker if you wish to use it
- All Tetris style blocks and Dr Mario Style blocks all have a 4x4 mini version used to display in the next coming piece window display, except the 1 bit version due to color limitations, but can substitute with the 8x8 tiles perfectly.
- Both Tetris and Dr Mario style games have full UI and play area, with full numbers and alphabet text/font, unless you have your own design or layout to use, which you are more than welcome to obviously.
- Each game style has an 'Options' and 'Pause' button.
- There is an optional display window to show your games title name or a logo, or anything else you wish, just keep in mind that you may not be able to sell your game using the Tetris or Dr Mario title name, so I would recommend creating your own game title name.

📌 Notes:
- To avoid tile confusion, there are single 4x4 and main 8x8 tiles separately, and also complete joined tile sets for easy grabbing and visual references, the main UI and play area also have complete and tiled grid for easy grabbing and layout.
- As mentioned with the Dr Mario 1bit style game, the mini 4x4 next piece display is impossible due to one color limitations, but you can easily use the 8x8 tiles instead, the main reason for 4x4 for colored tiles was the 4 blocks wide display, to keep things compact in the preview window and to not take the focus away from the play area.
- A reminder to just keep in mind that you may not be able to sell your game using the Tetris or Dr Mario title name, so I would recommend creating your own game title name.
